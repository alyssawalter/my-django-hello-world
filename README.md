# My Django Hello World Project

This is a simple Django project that returns a "Hello World!" message as a JSON response.



## Requirements


Before you begin, ensure you have met the following requirements:

- Python: You need to have Python installed on your system. Get more detail here: https://docs.djangoproject.com/en/4.2/faq/install/#faq-python-version-support

- Django: Make sure you have Django installed. You can install it using pip: pip install Django. 
Get more detail here: https://docs.djangoproject.com/en/4.2/intro/install/


## Installation

1. Clone this repository to your local machine:

```bash

git clone https://github.com/yourusername/my-django-hello-world.git

```

2. Navigate to the project directory:

```bash
 
cd my-django-hello-world

```


## Usage

1. Start the Django development server:

```bash

python manage.py runserver

```

2. Open your browser and access the JSON response at http://localhost:8000/hello/.

You should see a JSON response with the message: '{"Message": "Hello World!"}'. 

3. To stop the server, press 'Ctrl+C' in the terminal.
